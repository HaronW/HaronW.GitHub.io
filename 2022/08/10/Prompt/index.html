Prompt Learning

T5 11b para

seq2seq

GPT-3 175b

Description(Prompt)

zero-shot

few-shot

PLM Pre-trained Language Model

MLM Masked Language Model

bigger model, harder fine-tuning

bridge the gap between pre-trainging and fine-tuning

  • Add additional context (template) with a [MASK] position
  • Project labels to label words (verbalizer)

可以通过template 和 verbalizer 来调整需要进行的任务，使得一个模型可以完成多种任务

  • Pre-trained Model
    • Auto-regressive (GPT, OPT…) 生成
    • Masked Language Modeling (BERT, RoBERTa, DeBERTa…) 理解
    • Encoder-Decoder (T5, BART…)
  • Template
    • Manually Design
    • Auto Generation
    • Textual or Continuous…
  • Verbalizer
    • Manually Design
    • Expanding by external knowledge…

Auto-regressive

  • suitable for super-large pretrained models
  • Auto-regressive Prompt

Encoder-Decoder

  • Bidirectional attention for encoder
  • Auto-regressive for decoder

Template

  • Manully Design based no the characteristics of the task
  • Auto Generation with search or optimization
  • Textual or Continuous
  • Structured, incorporating with rules

template 有各种各样的形式
